1. Hiring Assistant AI – Agentic Resume Screening System (Academic Project)

This intelligent hiring assistant was built to transform traditional recruitment by integrating agentic AI with Retrieval-Augmented Generation (RAG). It autonomously screens and ranks resumes, engages recruiters through dynamic chat, and evolves based on continuous feedback—all in a privacy-first, offline-ready architecture.

Objective: To develop a lightweight, privacy-compliant, fully offline RAG-powered AI assistant that reduces hiring friction, enhances fairness, and adapts to evolving recruiter criteria in real-time.

Core Features:

  • Resume Parsing: PyMuPDF-based extraction from PDFs (resumes & job descriptions)
  • Semantic Matching: SentenceTransformers embeddings + in-memory similarity for context-rich matching
  • Hybrid Scoring: Combines semantic similarity with rubric-based skill match using Pandas
  • Recruiter Interaction: Dynamic conversations powered by quantized Mistral-7B-Instruct model
  • Feedback Loop: Updates scoring logic and rubric based on recruiter input

Technologies:

  • Python 3.x for backend logic and processing
  • Streamlit: Interface for recruiter interaction
  • PyMuPDF: For text extraction
  • SentenceTransformers: all-MiniLM-L6-v2 embeddings
  • Pandas: Skill rubric matching & hybrid ranking
  • ctransformers: For local inference using quantized Mistral-7B-Instruct
  • RAG: Integrated retrieval + reasoning for enhanced explainability
  • Offline Mode: Fully local, secure, and cloud-free operation

Key Outcomes:

  • Efficiency: Reduced time-to-hire by automating resume triage and ranking
  • Transparency: Provided clear explanations behind matches and rankings
  • Adaptability: Evolved in real-time based on recruiter input and feedback
  • Privacy-Focused: Local inference ensured no external dependencies

2. Visualizing Global Trends in Child Malnutrition (Academic Project)

Leveraging global health datasets from WHO, UNICEF, and the World Bank, this project offered a deep dive into malnutrition trends—analyzing stunting, wasting, underweight, and overweight metrics across geography, gender, and age. By transforming complex raw data into interactive dashboards, the project empowered global stakeholders to make informed decisions targeting child health inequalities.

Key Deliverables:

  • Malnutrition Severity Dashboard: Filterable by country, region, gender, and year—offering a 360° view of malnutrition prevalence
  • Child Patterns Dashboard: Trend analysis by age, visualized through scatter plots, time-series, box plots, and stacked area charts
  • Geospatial Heatmaps: Interactive mapping to highlight regional malnutrition disparities

Data Engineering & Insights:

  • Cleaned & standardized massive health datasets, resolving missing and inconsistent values
  • Mapped historical data (1970–2022) to expose evolving trends across urban-rural lines and genders
  • Identified critical hotspots like Papua New Guinea and Yemen demanding urgent intervention

Tools & Tech Stack:

  • Python (Pandas, NumPy): For data cleaning, transformation, and wrangling
  • Tableau / Power BI: To build interactive, high-impact dashboards
  • Excel / Google Sheets: Initial validation and exploration
  • Visualization Techniques: Geospatial mapping, trend & comparative analysis, interactive filters

3. AI-Powered Fraud Detection in Banking (Academic Project)

This project envisioned and specified a real-time AI-driven fraud detection system tailored for the financial sector. Replacing traditional rule-based methods, the system leverages machine learning models for proactive fraud prediction, pattern recognition, and compliance-oriented auditing.

Core Features:

  • Real-Time Monitoring: Flags suspicious behavior with latency under 300ms
  • Supervised & Unsupervised ML: Detects both known and emerging fraud tactics
  • Investigation Module: Analysts can review, annotate, and escalate flagged cases with feedback loops
  • Risk Scoring: Behavioral analysis scores each transaction dynamically, with instant alerts via dashboards & SMS
  • Explainable AI: SHAP/LIME ensures transparent, auditable decisions for compliance (GDPR, AML, etc.)

Technical Highlights:

  • Continuous Learning: Adaptive model retraining using analyst feedback
  • High Availability: 99.99% uptime with scalable, containerized microservices
  • Security Protocols: AES-256, TLS 1.3, MFA, RBAC implemented
  • Ethical AI: Bias mitigation and fair decision logic integrated across all demographic groups

Use Cases & Value:

  • Real-Time Fraud Workflow: Full lifecycle from transaction monitoring to analyst resolution
  • Operational Efficiency: Reduces fraud losses and strengthens user trust with predictive insights
  • Regulatory Compliance: Ensures conformance to GDPR, PCI DSS, KYC, AML, and CCPA standards
