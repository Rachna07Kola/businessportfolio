Projects

1. Hiring Assistant AI – Agentic Resume Screening System

Objective: A lightweight, privacy-compliant, offline RAG-powered AI assistant that streamlines hiring while ensuring fairness.

  • Resume Parsing: PyMuPDF-based extraction
  • Semantic Matching: SentenceTransformers embeddings + in-memory similarity
  • Hybrid Scoring: Combines embeddings with rubric logic using Pandas
  • Recruiter Interaction: Powered by quantized Mistral-7B-Instruct via ctransformers
  • Feedback Loop: Recruiter-driven adaptive ranking and logic updates

Technologies: Python, Streamlit, PyMuPDF, SentenceTransformers, Pandas, ctransformers, RAG

Key Outcomes: Automated triage, transparency in scoring, offline secure ops, adaptive feedback.
